# CodeAlpha Basic Network Sniffer

## Description
This project is a Python-based network sniffer developed as part of the CodeAlpha Cyber Security Internship.
It captures live network packets and displays source IP, destination IP, protocol, and port information.

## Technologies Used
- Python
- Scapy Library

## Features
- Captures live network traffic
- Detects TCP, UDP, and ICMP packets
- Displays packet details in real time

## How to Run
1. Install scapy: pip install scapy
2. Run as administrator
3. Execute: python sniffer.py

## Learning Outcome
This project helps understand network protocols, packet structure, and real-time traffic analysis.
